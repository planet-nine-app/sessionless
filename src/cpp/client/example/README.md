# Sessionless C/C++ Example

## Dependencies 
* [sessionless](https://github.com/planet-nine-app/sessionless)

    > _**NOTE:**_ when using CMake the dependencies are downloaded automatically

## Evironment setup 
CMake and C++ compiler are required.

## Build the example
```shell
mkdir build
cmake -B ./build -S .
cd build
make
```

## Run the example
```shell
cd build
./example
```
